Buatkan landing page modern dan responsif untuk aplikasi web "Sistem Informasi Pemesanan Tiket Wisata" menggunakan HTML5, CSS3, Bootstrap 5, JavaScript, dan desain UI/UX profesional.

Tema desain:

* Warna utama: Biru (#0D6EFD), Hijau (#198754), Putih (#FFFFFF)
* Tampilan modern, clean, dan user friendly
* Menggunakan efek animasi ringan (AOS atau CSS Animation)
* Mobile responsive
* Menggunakan icon Bootstrap Icons atau Font Awesome

Landing page harus terdiri dari section berikut:

1. Navbar

* Logo Sistem Informasi Pemesanan Tiket Wisata
* Menu: Beranda, Destinasi, Fitur, Galeri, Tentang, Kontak
* Tombol Login dan Daftar

2. Hero Section

* Background gambar wisata Kalimantan Barat
* Judul utama:
  "Jelajahi Destinasi Wisata Terbaik Kalimantan Barat"
* Deskripsi singkat sistem
* Tombol:

  * Pesan Tiket Sekarang
  * Lihat Destinasi

3. Section Tentang Sistem

* Penjelasan mengenai Sistem Informasi Pemesanan Tiket Wisata
* Menampilkan ilustrasi atau gambar wisata
* Menjelaskan kemudahan pemesanan tiket secara online

4. Section Destinasi Wisata Unggulan
   Tampilkan dalam bentuk card dengan gambar, nama destinasi, lokasi, dan tombol detail.

Contoh destinasi:

* Tugu Khatulistiwa
* Pantai Pasir Panjang Singkawang
* Danau Sentarum
* Keraton Kadariah Pontianak
* Bukit Kelam Sintang
* Pulau Lemukutan

5. Section Fitur Sistem
   Tampilkan dalam bentuk card icon.

Fitur:

* CRUD Data Destinasi Wisata
* CRUD Data Pengunjung
* CRUD Kategori Wisata
* Pemesanan Tiket Online
* Upload Foto Destinasi
* Laporan Pemesanan
* Dashboard Statistik
* Sistem Login dan Hak Akses

6. Section Alur Pemesanan
   Buat step-by-step:

7. Pilih Destinasi

8. Isi Data Pengunjung

9. Pilih Jumlah Tiket

10. Lakukan Pemesanan

11. Cetak Tiket

12. Section Statistik
    Tampilkan counter animasi:

* Total Destinasi
* Total Pengunjung
* Total Pemesanan
* Total Kategori Wisata

8. Section Galeri Wisata
   Grid gambar wisata Kalimantan Barat dengan efek hover modern.

9. Testimoni Pengunjung
   Minimal 3 testimoni dalam bentuk carousel.

10. Call To Action
    Judul:
    "Siap Berwisata di Kalimantan Barat?"
    Tombol:
    "Pesan Tiket Sekarang"

11. Footer

* Informasi sistem
* Link cepat
* Kontak
* Media sosial
* Copyright

Kriteria tambahan:

* Gunakan Bootstrap Grid System
* Gunakan komponen card modern dengan shadow dan hover effect
* Gunakan animasi fade-up saat scroll
* Struktur kode rapi dan mudah dikembangkan menjadi project CodeIgniter 4
* Pisahkan file HTML, CSS, dan JavaScript
* Sertakan komentar pada setiap section agar mudah dipahami developer
* Tambahkan dummy data dan placeholder gambar wisata Kalimantan Barat
* Optimasi SEO dasar menggunakan meta title, meta description, dan favicon
* Hasil akhir harus terlihat seperti website pariwisata profesional dan siap diintegrasikan dengan backend CodeIgniter 4.
